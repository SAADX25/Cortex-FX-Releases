# Cortex FX 🚀
**The Ultimate Media Converter for Windows**

Cortex FX is a modern, high-performance desktop application designed to convert Images, Videos, Audio, and Documents with a simple Drag & Drop interface. Built with .NET 8 and WPF.

![Cortex FX Screenshot]

## ✨ Features
- **Modern UI:** Dark mode, chromeless window, and smooth animations.
- **Drag & Drop:** Simply drop your files to convert them instantly.
- **All-in-One:** - 🖼️ **Images:** Convert to PNG, JPG, WEBP, BMP.
  - 🎥 **Video:** Convert to MP4, AVI, MKV, GIF.
  - 🎵 **Audio:** Convert to MP3, WAV, M4A.
  - 📄 **Documents:** (New!) Convert Word & PowerPoint to PDF.
- **Privacy Focused:** Everything runs locally on your device (Offline). No cloud uploads.

- **Installation:** No external dependencies required (Self-Contained).
